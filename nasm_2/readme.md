# NASM_2 Программа вывода "Hello world!"

## Задача - вывести в терминале cmd текстовое сообщение

Для подключения внешних функций из библиотеки kernel32 надо добавить "-l kernel32":

    в Makefile изменить в строке 25
    $(CC) $(OBJ) -o $(TARGET) -l kernel32

Написать программу и запустить в терминале VS Code команду:

    make

Запустить программу в терминале cmd:

    диск:\путь\main.exe

В консоли должно появиться сообщение:

    Hello world!

---
После тестирования программы можно очистить проект:

    make clean
