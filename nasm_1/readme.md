# NASM_1

## Задача

вывести в консоль строку - Hello world!

использовать сборку через make

    make

---
Проверка программы в терминале cmd:

    >путь/main.exe

В консоле ничего не произойдет, надо ввести команду и она должна вывести число 22:

     >echo %ERRORLEVEL%
     >22

---
После тестирования программы можно очистить проект:

    make clean
