# NASM_1 Программа установки значения в регистр RAX

## Задача 1 - установить в регистр RAX значение 22

Написать первую самую простую программу и создать запускающий .EXE-файл

Для сборки использовать NASM и MAKE, команды:

    nasm -f win64 main.asm -o main.obj
    ld main.obj -o main.exe

Запустить программу в терминале cmd:

    диск:\путь\main.exe

Программа ничего не выводит на консоль, но устанавливает состояние завершения в значинии 22,
чтобы это проверить необходимо сразу после main.exe выполнить команду в терминале cmd:

    %ERRORLEVEL%

В терминале должно появиться число 22

Удалить файлы .exe и .obj

## Задача 2 - создать и использовать Makefile

Вместо команд nasm и make надо настроить Makefile для автоматической сборки и
запустить в терминале VS Code команду:

    make

---
После тестирования программы можно очистить проект:

    make clean
